### 导入
> * 手动导入：
>
>> * **SwpTools** 文件夹导入项目中。
>> * **`#import  SwpTools.h`**

> -------

> * CocoaPods 导入:
>
>> * **pod search SwpRequest**
>> * **pod 'SwpTools'**
>> * **`#import <SwpTools/SwpTools.h>`**
